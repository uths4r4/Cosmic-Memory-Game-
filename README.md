# Cosmic Memory Game

A fast, cosmic-themed **memory matching** game built with **HTML, CSS, and JavaScript**. Flip cards, remember their positions, and match all pairs as quickly as you can.

## Play the Game
- **Local:** download/clone the repo and open `index.html` in your browser.
- **GitHub Pages (optional):** if you enable Pages, add the deployed link here:  
  `https://uths4r4.github.io/Cosmic-Memory-Game-/`

## Features
- Cosmic UI theme
- Card-flip matching gameplay
- Move/attempt tracking (if enabled in your script)
- Quick restart (if enabled)
- Responsive layout for desktop/mobile

## How to Play
1. Start/restart the game.
2. Click (or tap) a card to flip it.
3. Flip a second
